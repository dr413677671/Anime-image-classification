## About this project

Anime theme classification.

One stage classification (home-trained) + combined background matting.

<img src='./illustration_preview.png'>

## Features:

- [x] Model
    - [x] Xception
    - [ ] SWIN2 (TBD)
- [x] Data Augmentation
    - [x] Random Crop
    - [x] Rescale
    - [x] Flip
    - [ ] Saturation
    - [ ] Contrast
    - [ ] Noise/Denoise
    - [ ] Mild rotate
    - [ ] Mask
- [ ] Hyperparameter Search
- [ ] Ensemble Learning
- [x] Others
    - [x] Early-stopping
    - [x] Finetune
    - [x] Optimizer-Adam
    - [x] Multi-labels classification
    - [ ] Class-weighting
