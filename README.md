## About this project

Anime image theme classification. Home-trained model. Will open-source training and models later.

## Support:
- [x] Model
    - [x] Xception
    - [ ] SWIN2 (TBD)
- [x] Data Augmentation
    - [x] Random Crop
    - [x] Rescale
    - [x] Flip
    - [ ] Saturation
    - [ ] Contrast
    - [ ] Noise/Denoise
    - [ ] Mild rotate
    - [ ] Mask
- [ ] Hyperparameter Search
- [ ] Ensemble Learning
- [x] Others
    - [x] Early-stopping
    - [ ] Finetune
